# Personal_finance_android
Local-first personal finance app (Android, Kotlin, Room)

## Purpose
A personal finance app to track salary, expenses, and savings ratio.
Local-first design with no server storage.

## Key Principles
- No bank login, no server
- Fully local database (Room)
- Encrypted backup & restore
- Privacy-first

## Tech Stack
- Kotlin
- Jetpack Compose
- Room (SQLite)
- Android Biometric API
